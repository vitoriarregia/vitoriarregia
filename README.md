### 📝 Sobre o Projeto
Como aspirante a **Analista de Dados**, desenvolvi esta solução para resolver o problema de preenchimento manual e repetitivo de informações. O bot automatiza a leitura de uma base de dados estruturada e realiza o input direto no sistema web, garantindo integridade e agilidade no processo de gestão de dados.

### ⚠️ Aviso Técnico: Coordenadas de Tela
Este projeto utiliza a biblioteca **PyAutoGUI** com coordenadas de clique fixas no código:
* **Resolução**: O script foi configurado com base na resolução do meu monitor atual.
* **Portabilidade**: Para rodar em outras máquinas, é necessário atualizar as coordenadas `x` e `y` no arquivo `Codigo.py` para os campos específicos da sua tela.

---

### 🚀 Fluxo de Execução
1. **Login Automatizado**: Acesso ao portal e preenchimento de credenciais via teclado simulado.
2. **Manipulação com Pandas**: Leitura do arquivo `produtos.csv` para estruturação dos dados.
3. **Loop de Cadastro**: Iteração automática para preencher campos de código, marca, tipo e valores.
4. **Navegação Dinâmica**: Comandos de scroll para garantir a visibilidade dos campos durante o processo.

---

### 🛠️ Stack Tecnológica
* **Python**: Linguagem principal para lógica de automação.
* **Pandas**: Manipulação e análise da base de dados.
* **PyAutoGUI**: Controle de interface humana (RPA).
* **Openpyxl**: Suporte para integração de arquivos.
